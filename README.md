# FUTURE_DS_01 – Business Sales Performance Analytics

## Project Overview

This project focuses on analyzing business sales performance using Power BI. The objective is to identify sales trends, top-performing products, regional performance, and category-level insights to support data-driven business decisions.

## Tools Used

* Power BI
* Microsoft Excel
* GitHub

## Dataset

The dataset contains sales transaction records, including:

* Order Date
* Region
* Category
* Product Name
* Sales
* Profit
* Quantity

## Dashboard Features

The dashboard includes:

* Total Sales KPI
* Total Profit KPI
* Sales Trend Analysis
* Regional Sales Performance
* Category Revenue Analysis
* Top Selling Products
* Interactive Region Filter

## Key Findings

* Technology generated the highest sales revenue.
* The West region recorded the strongest sales performance.
* Sales trends varied across different periods.
* A small number of products contributed significantly to overall revenue.

## Recommendations

1. Increase investment in high-performing product categories.
2. Expand marketing efforts in top-performing regions.
3. Monitor top-selling products to maintain stock availability.
4. Develop strategies to improve lower-performing categories.
